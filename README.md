# Bob's External

A Windows Forms application built with .NET 8.

## Features

- Dark theme UI with sidebar navigation
- Dashboard with configurable settings
- Feature toggle and opacity control
- Theme selector (Dark/Light)

## Build

To build the project in Release mode:

```bash
dotnet build -c Release
```

## Run

To run the application:

```bash
dotnet run
```

## Output

Release build output will be in: `bin/Release/net8.0-windows/`
